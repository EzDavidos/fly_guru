# Чек-лист ручной проверки FlyGuru

Проходишь галочки по порядку. Слева меню кабинета — идём сверху вниз по нему.
Держи под рукой два аккаунта: **админ** и **инструктор** (см.
[commands.md](./commands.md)).

Условные обозначения: ⬜ — проверить, ⚠️ — типовая точка отказа.

---

## Админка (`/admin`)

### Заявки
- ⬜ Заявка с сайта появляется в списке.
- ⬜ «Записать клиента» из заявки переносит имя/телефон в форму.
- ⬜ Отклонённая/обработанная заявка уходит из активных.

### Записать клиента
- ⬜ Новый клиент создаётся; повторный телефон НЕ плодит вторую карточку.
- ⬜ Услуга, сумма и формат оплаты обязательны — без них не сохраняет.
- ⬜ Скидка −10% по коду агента применяется только к базовому обучению.

### Календарь
- ⬜ Смены инструкторов видны по дням; в ячейке — имя назначенного инструктора.
- ⬜ Переключение месяцев работает.

### Сессии
- ⬜ Записанное занятие появляется в списке с суммой и инструктором.
- ⬜ Удаление сессии работает и пересчитывает статистику.

### Абонементы
- ⬜ Проданный абонемент виден, остаток минут считается верно.
- ⬜ При истечении видно, чьи минуты сгорели.
- ⬜ Удаление абонемента работает.

### Клиенты
- ⬜ Поиск/список клиентов, карточка открывается.
- ⬜ Видно, кто из лидов агента оплатил, а кто только записался.

### Агенты
- ⬜ Создание агента, реф-код генерится.
- ⬜ Начисленные награды видны.

### Члены клуба
- ⬜ Список членов, скидка/статус корректны.

### Материалы
- ⬜ Материалы отображаются/добавляются.

### Статистика
- ⬜ Цифры за месяц сходятся (выручка, новые клиенты, сессии).
- ⬜ Широкий ПК-лейаут не разъезжается.

### Расчёт месяца (payroll)
- ⬜ ЗП инструктора: 200к/выход + 15% с чека минус комиссия агента.
- ⬜ Расходы марины (35%) учтены.

### Расходы
- ⬜ Админский расход добавляется/удаляется.
- ⬜ Расходы инструктора падают сюда же.

### Услуги
- ⬜ Цена/название правятся, на сайте цены тянутся из базы.

### Настройки
- ⬜ Инвентарь (доски/крылья) добавляется — ⚠️ без него у инструктора не с чем
  связать фото смены.
- ⬜ Справочники (оплаты, категории расходов) редактируются.

## Кабинет инструктора (`/instructor`)

Заходи под аккаунтом инструктора (НЕ под админом — у него другой кабинет).

### Записи
- ⬜ Подтверждённая заявка видна; «Принять» закрепляет за собой, «Отказаться»
  возвращает в пул.

### Записать клиента
- ⬜ Подсказка «этот клиент уже у нас» всплывает по телефону ДО отправки.
- ⬜ Занятие записывается, экран «Готово» показывает сумму.
- ⬜ Личная реф-ссылка создаётся и копируется.

### Смена ⚠️ (главный сценарий этой правки)
- ⬜ Утром: выбрать доску из списка → выбрать фото → **фото засчитывается сразу,
  без нажатия «+»**.
- ⬜ То же для крыла.
- ⬜ «Связь» и «Дефект» спрятаны в блоке **«Дополнительно»** (стрелка-раскрытие).
- ⬜ Кнопка «Открыть смену» разблокируется, когда есть доска + крыло.
- ⬜ После открытия появляется секция «Закрытие».
- ⬜ Вечером: так же добавить фото доски/крыла для закрытия — ⚠️ раньше тут
  фото «не добавлялось», проверь что теперь засчитывается сразу.
- ⬜ «Закрыть смену» срабатывает; статусы времени (вовремя/поздно/залёт) верны.
- ⬜ Незапланированный выход помечается («админ смены не ставил»).

### Календарь
- ⬜ Свои смены видны по дням.

### Статистика
- ⬜ Свои цифры за месяц; допуск клиента к выездам ставится.

### Абонемент
- ⬜ Продажа абонемента: клиент становится членом клуба, отметка оплаты работает.

### Списание
- ⬜ Списание минут не даёт уйти в минус; при нуле абонемент помечается «использован».
- ⬜ Истёкший абонемент не даёт списывать.

### Расходы
- ⬜ Свой расход добавляется/удаляется; чужие не видны.

### Настройки
- ⬜ Имя, возраст, цель по ЗП сохраняются; аватар загружается.

---

## Быстрая проверка ролей
- ⬜ Инструктор не может открыть `/admin` — редиректит в свой кабинет.
- ⬜ После `set-role` и перелогина роль применилась (нет рассинхрона JWT/БД).
