Это чек-лист всей информации, контента и ссылок, которые нужны, чтобы заменить
заглушки на реальные данные. Заполняй прямо в этом файле (пиши ответ после каждого
пункта) или скидывай мне пакетом — я разнесу по коду.

Приоритеты:
- 🔴 **СЕЙЧАС** — блокирует запуск живого сайта, стоит заглушка на видном месте.
- 🟡 **СКОРО** — нужно для полноты, но сайт уже работает без этого.
- 🟢 **ПОТОМ** — для будущих этапов (магазин, кабинеты), собрать заранее не помешает.

## 3. 🔴 Отзывы клиентов
*Где используется: блок на главной (2–3 лучших) + страница `/reviews` (все).
Файл: `src/content/reviews.ts`. Сейчас 6 штук — все рыба.*

Для каждого отзыва нужно:
- **Имя** (можно только имя или имя + первая буква фамилии)
- **Город / откуда** (напр. «Москва», «Новосибирск», «Ханой»)
- **Текст отзыва** (реальные слова клиента)
- **(опц.) Фото** клиента или его полёта — сильно повышает доверие
- **(опц.) Ссылка на источник** (Instagram/Google-отзыв) для достоверности

> Скинь сколько есть реальных — минимум 2–3 для главной. Остальные заменим по мере
> появления. Если есть скрины отзывов из мессенджеров/Instagram — тоже подойдут,
> перепечатаю.

## 4. 🔴 Фото и видео
*Сейчас везде стоит серый SVG-плейсхолдер. Список мест, где нужны реальные медиа
(в порядке важности):*

- [ ] **Главная, первый экран** — эффектное фото/видео полёта над водой
      (это первое, что видит гость). Идеально — короткое видео. Файл: `page.tsx`.
- [ ] **Обучение** (`/training`) — фото ученика на воде с инструктором рядом.
- [ ] **Тандем** (`/tandem`) — фото/видео тандемного полёта (двое на доске).
- [ ] **Реф-лендинг** (`/r/[code]`) — фото/видео довольного гостя на воде.
- [ ] **Магазин / тизер на главной** — фото фойла (лучше на белом/чистом фоне).
- [ ] **(опц.) Клуб, сафари/путешествия** — фото с выездов, если есть.

Технически по медиа:
- [ ] Формат: фото — JPG/PNG/WebP; видео — MP4 (лучше без звука, зациклённое).
- [ ] Ориентация: желательно и вертикальные (телефон), и горизонтальные.
- [ ] Права: это ваши материалы / есть разрешение людей в кадре? (да/нет)
- [ ] Как удобнее передать: Google Drive / Telegram / прямо в чат? _______
